Portfolio

TurnIt

The TurnIt is a Bluetooth motion-activated bicycle indicator light system that uses an Arduino microcontroller and Pebble smart-watch. The Pebble watch accelerometer detects various arm gestures to control corresponding bike turn signal lights. The lighting system consists of a five-segment LED arrow array embedded into a 3-D printed electronics housing using Solidworks.

CodePaper

CodePaper is a graphical programming language which interprets user-drawn flow diagrams for functional programming! It utilizes the Python OpenCV computer vision library for image processing and a unique a binary search tree algorithm to identify nodes and edges. Also includes a web application for users to submit and process photos of hand-drawn diagrams.

Serial Protocol Parser

This serial protocol parser was developed for the Waterloo Hybrid SAE racing team. It is an engine data parsing software that transforms raw data from hybrid race car vehicle control unit into analyzable information. Along with the parser, an Arduino microcontroller was also used to create a testing program that streamed pseudo-data from serial for car engine simulation.
